백엔드 DB Import : 
1. 워크벤치에서 File- Open SQL Script 클릭
2. /I-Run-You/Back/db-import/i_run_you.sql 선택
3. Ctrl + Shift + Enter로 전체 구문 실행
4. 스키마 새로고침해서 확인 가능

백엔드 이클립스 Import :
1. 이클립스에서 Import - Gradle - Exsting Gradle Project 에 들어감
2. Next 누르고 Project root directory 를 /I-Run-You/Back/spring-boot 로 설정
3. Next 누르고 Override workspace settings 체크
4. Finish 눌러서 Import
src-com.project.irunyou-UrunyouApplication.java 실행시키면 서버 실행됨
